# gamedesign week 1

## opdracht

Tijdens dit project ga je alle losse elementen voor een spel maken. Tijdens deze week bedenk je welk spel je wilt (na)maken. Je schrijft daarbij een tekst die gedetailleerd beschrijft:

 - welke objecten (spelers, tegenstanders, dingen, blokken, ...) er allemaal voorkomen in het spel
 - hoe die objecten zich gedragen
